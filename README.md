
## Installation
in order to run it we need docker and docker compose to start this project
also update the .env in with your own mongoURL

## Running the app

```bash
# development
$ docker compose up --build 

```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
